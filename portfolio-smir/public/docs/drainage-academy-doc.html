1. Contexte et objectifs

Drainage Academy est une ecole professionnelle de drainage lymphatique fondee par Dan Samama. La plateforme complete comprend un site vitrine public, un espace praticien securise et un back-office d'administration complet.

L'objectif etait de developper une solution fullstack (front + back + middleware) permettant au fondateur de gerer de maniere autonome l'integralite du contenu du site depuis une interface d'administration intuitive.

2. Architecture technique

Frontend (Site public)

  • React : Interface utilisateur moderne et reactive
  • Design responsive mobile-first
  • Palette professionnelle beige/neutre
  • Carrousel de temoignages video avec auto-defilement
  • Integration WhatsApp pour les consultations

Middleware (Authentification)

  • Systeme d'authentification securise pour les praticiens
  • Gestion des sessions avec tokens JWT
  • Roles et permissions (admin, praticien, visiteur)
  • Protection des routes sensibles

Backend (Administration)

  • Node.js : API REST pour la gestion de contenu
  • Base de donnees : Stockage des praticiens, articles, avis, medias
  • Interface d'administration complete pour Dan (le fondateur)
  • Upload et gestion de fichiers (photos, videos)

5. Espace Praticien

  • Authentification securisee : Login par email/mot de passe avec tokens JWT
  • Profil personnel : Modification des informations, photo, specialites
  • Ressources exclusives : Acces aux videos de formation, supports pedagogiques
  • Annuaire : Apparition automatique dans le repertoire public apres validation admin

6. Securite

  • Authentification JWT avec expiration et refresh tokens
  • Hashage des mots de passe (bcrypt)
  • Validation de toutes les entrees utilisateur (cote serveur)
  • Protection CORS et rate limiting sur l'API
  • Upload de fichiers securise (type MIME, taille max)
  • Roles et permissions granulaires (admin / praticien / public)

7. Formation proposee par Drainage Academy

Programme en 3 piliers

  1. Le Geste : Technique precise de drainage lymphatique, correction individualisee en presentiel
  2. L'Apprentissage : Formation theorique en video, accessible a vie, revisable a volonte
  3. La Pratique : 2 jours intensifs en petit groupe (4 personnes max)

Accompagnement business inclus

  • Visibilite locale : optimisation de la presence en ligne
  • Fidelisation client : strategies de retention
  • Structure d'activite : aide a la creation d'entreprise
