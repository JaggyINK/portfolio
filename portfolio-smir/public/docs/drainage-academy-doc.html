1. Contexte et objectifs

Drainage Academy est une école professionnelle de drainage lymphatique fondée par Dan Samama. La plateforme complète comprend un site vitrine public, un espace praticien sécurisé et un back-office d'administration complet.

L'objectif était de développer une solution fullstack (front + back + middleware) permettant au fondateur de gérer de manière autonome l'intégralité du contenu du site depuis une interface d'administration intuitive.

2. Architecture technique

Frontend (Site public)

  • React : Interface utilisateur moderne et réactive
  • Design responsive mobile-first
  • Palette professionnelle beige/neutre
  • Carrousel de témoignages vidéo avec auto-défilement
  • Intégration WhatsApp pour les consultations

Middleware (Authentification)

  • Système d'authentification sécurisé pour les praticiens
  • Gestion des sessions avec tokens JWT
  • Rôles et permissions (admin, praticien, visiteur)
  • Protection des routes sensibles

Backend (Administration)

  • Node.js : API REST pour la gestion de contenu
  • Base de données : Stockage des praticiens, articles, avis, médias
  • Interface d'administration complète pour Dan (le fondateur)
  • Upload et gestion de fichiers (photos, vidéos)

4. Espace Administration (Back-office)

Le back-office est le cœur du projet : il permet à Dan de gérer l'intégralité du contenu visible sur le site public, sans aucune compétence technique requise.

5. Espace Praticien

  • Authentification sécurisée : Login par email/mot de passe avec tokens JWT
  • Profil personnel : Modification des informations, photo, spécialités
  • Ressources exclusives : Accès aux vidéos de formation, supports pédagogiques
  • Annuaire : Apparition automatique dans le répertoire public après validation admin

6. Sécurité

  • Authentification JWT avec expiration et refresh tokens
  • Hashage des mots de passe (bcrypt)
  • Validation de toutes les entrées utilisateur (côté serveur)
  • Protection CORS et rate limiting sur l'API
  • Upload de fichiers sécurisé (type MIME, taille max)
  • Rôles et permissions granulaires (admin / praticien / public)

7. Formation proposée par Drainage Academy

Programme en 3 piliers

  1. Le Geste : Technique précise de drainage lymphatique, correction individualisée en présentiel
  2. L'Apprentissage : Formation théorique en vidéo, accessible à vie, révisable à volonté
  3. La Pratique : 2 jours intensifs en petit groupe (4 personnes max)

Accompagnement business inclus
