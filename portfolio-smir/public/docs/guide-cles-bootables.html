1. Comprendre la clé USB bootable

Une clé USB bootable est un support amovible capable de démarrer un ordinateur indépendamment du système d'exploitation installé sur le disque dur. Elle contient tout le nécessaire pour lancer un OS, un outil de réparation ou un programme de diagnostic.

Composition d'une clé bootable

  • Bootloader — Le programme d'amorçage qui s'exécute en premier au démarrage. Il indique au BIOS/UEFI comment charger le système contenu sur la clé.
  • Structure compatible BIOS/UEFI — La clé doit être formatée avec un système de fichiers reconnu par le firmware de la carte mère. Le FAT32 est le format le plus universellement compatible avec l'UEFI.
  • Image système (ISO) — Le fichier .iso contient l'intégralité du système d'exploitation ou de l'outil à démarrer, compressé dans une archive standardisée.

Une image ISO doit être « gravée » (burnée) sur la clé USB à l'aide d'un outil spécialisé. Copier-coller le fichier .iso directement sur la clé ne la rendra pas bootable. Le processus de gravure extrait le contenu de l'ISO et configure le bootloader nécessaire.

2. Formats et leur rôle

Le choix du système de fichiers lors du formatage de la clé USB a un impact direct sur la compatibilité et les capacités de la clé bootable.

Format Compatibilité UEFI Limite fichier Cas d'usage
FAT32 Maximale (Windows, Linux, macOS, BIOS, UEFI) Oui 4 Go max par fichier Choix par défaut pour toute clé bootable UEFI
NTFS Windows principalement Limitée Aucune limite pratique ISO Windows > 4 Go (install.wim volumineux)
exFAT Mac & Windows Pas toujours Aucune limite pratique Échange de fichiers Mac/PC, rarement bootable

FAT32 = choix le plus sûr pour l'UEFI. En cas de doute, formatez toujours en FAT32. Rufus gère automatiquement le fractionnement des fichiers > 4 Go si nécessaire.

4. Créer sous Windows avec Rufus

Procédure pas à pas

1

Télécharger Rufus

Récupérez la dernière version sur rufus.ie. L'outil est portable (pas d'installation nécessaire).

2

Brancher la clé USB

Insérez votre clé USB (minimum 8 Go recommandé). Toutes les données seront effacées.

3

Sélectionner l'image ISO

Dans Rufus, cliquez sur « Sélection » et choisissez votre fichier .iso (Windows, Ubuntu, etc.).

4

Configurer les options

Sélectionnez le schéma de partition GPT et le système cible UEFI (non-CSM). Laissez le système de fichiers sur FAT32 (ou NTFS si l'ISO dépasse 4 Go).

5

Lancer la gravure

Cliquez sur « Démarrer ». Confirmez l'effacement de la clé. Patientez quelques minutes.

Pour les ISO Windows 11, Rufus propose des options pour contourner les vérifications TPM 2.0 et Secure Boot. Cochez ces options si vous installez sur du matériel ancien.

5. Créer sous macOS

Méthode 1 : BalenaEtcher (simple)

Téléchargez BalenaEtcher depuis etcher.balena.io, puis :

1

Flash from file — Sélectionnez votre fichier ISO

2

Select target — Choisissez votre clé USB

3

Flash! — Lancez la gravure et patientez

Méthode 2 : Terminal avec dd (avancé)

Pour les utilisateurs à l'aise avec le terminal :

Terminal — Identifier le disque 
diskutil list

Repérez votre clé USB (par exemple /dev/disk2). Vérifiez la taille pour confirmer.

Terminal — Démonter le disque 
diskutil unmountDisk /dev/disk2
Terminal — Graver l'ISO 
sudo dd if=/path/file.iso of=/dev/rdisk2 bs=1m

Utilisez /dev/rdisk2 (avec le préfixe r) pour une vitesse d'écriture bien supérieure.

La commande dd écrase intégralement le disque cible sans aucune confirmation. Une erreur sur le numéro de disque (par ex. /dev/disk0 au lieu de /dev/disk2) peut détruire le système d'exploitation de votre Mac. Vérifiez trois fois le disque cible avant d'exécuter la commande.

6. Clé multi-boot avec Ventoy

Ventoy est la solution idéale pour créer une clé universelle multi-ISO. Le principe est radicalement différent des outils classiques :

1

Installer Ventoy une seule fois

Téléchargez Ventoy depuis ventoy.net et installez-le sur votre clé USB. Cette opération formate la clé et installe le bootloader Ventoy.

2

Copier-coller les ISO

Ensuite, il suffit de glisser-déposer vos fichiers .iso directement sur la clé USB, comme sur une clé ordinaire. Aucun outil de gravure nécessaire.

3

Démarrer et choisir

Au boot, Ventoy affiche un menu listant toutes les ISO présentes. Sélectionnez celle que vous souhaitez lancer.

Aucun reformatage nécessaire pour ajouter ou supprimer des ISO. Ajoutez ou retirez simplement les fichiers. C'est la solution idéale pour les techniciens et les clés de réparation multi-outils.

8. Vérification

Après avoir créé votre clé bootable, vérifiez qu'elle fonctionne correctement :

1

Redémarrer le PC avec la clé USB branchée.

2

Accéder au Boot Menu en appuyant sur la touche correspondante dès l'affichage du logo constructeur.

3

Sélectionner la clé USB dans la liste des périphériques de boot.

4

Vérifier que l'ISO démarre correctement (menu d'installation, bureau live, etc.).

Touches Boot Menu par constructeur

Constructeur Touche Boot Menu
Dell F12
HP F9
Lenovo F12
Acer F12
ASUS F8 ou Esc
MSI F11

Si la touche ne fonctionne pas, essayez d'accéder au BIOS/UEFI (généralement F2 ou Suppr) et modifiez l'ordre de boot pour placer l'USB en premier.

9. Dépannage

« No bootable device found »

  • Activer le mode UEFI dans le BIOS — Certains PC anciens sont configurés en mode Legacy/CSM uniquement.
  • Désactiver le Secure Boot — Le Secure Boot peut rejeter les clés non signées. Désactivez-le temporairement dans le BIOS.
  • Reformater en FAT32 — Si la clé est en NTFS ou exFAT, reformatez-la en FAT32 et recommencez la gravure.

L'ISO ne démarre pas / écran noir

  • Gravure corrompue — L'ISO a peut-être été mal gravée. Refaites l'opération avec BalenaEtcher ou Rufus.
  • ISO endommagée — Vérifiez le hash SHA256 du fichier ISO téléchargé et comparez-le avec celui du site officiel.
  • Mauvais mode de boot — Assurez-vous que le mode (UEFI vs Legacy) correspond au schéma de partition utilisé (GPT vs MBR).

La clé USB n'est pas reconnue

  • Essayer un autre port USB — Privilégiez les ports USB 2.0 (noirs) pour le boot, certains firmwares ne supportent pas bien l'USB 3.0 au démarrage.
  • Reformater la clé — Utilisez diskpart (Windows) ou diskutil (macOS) pour nettoyer et reformater complètement la clé.
  • Tester avec une autre clé — La clé USB elle-même peut être défectueuse. Testez avec un autre modèle.

Avant toute manipulation BIOS/UEFI, notez les paramètres actuels. Un mauvais réglage peut empêcher le démarrage normal de votre système.

10. Conclusion

Savoir créer une clé USB bootable est une compétence essentielle de maintenance informatique. Que ce soit pour installer un système d'exploitation, réparer un PC qui ne démarre plus, ou lancer un diagnostic matériel, la clé bootable reste l'outil de premier recours du technicien.

En résumé :

  • Rufus — La référence pour créer une clé bootable simple sous Windows.
  • BalenaEtcher — Le choix le plus accessible, multi-plateforme.
  • Ventoy — La meilleure solution pour une clé multi-ISO : installez-le une fois, puis glissez-déposez autant d'ISO que la capacité de la clé le permet.

Recommandation : Pour un technicien ou un passionné d'informatique, la combinaison Ventoy + une clé USB de 64 Go constitue la trousse à outils idéale. Stockez-y Windows, une distribution Linux, Hiren's BootCD et Memtest86 — vous serez prêt à toute éventualité.
